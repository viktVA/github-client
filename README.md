# GitHub Client

SPA-приложение для просмотра публичных репозиториев крупных российских IT-компаний через GitHub API

## Стек
- React 18 + TypeScript
- Vite
- React Router v7
- SCSS + CSS Modules


## Функциональность

### Список репозиториев
- Параллельная загрузка репозиториев пяти организаций: Яндекс, Т банк, Авито, Wildberries, Озон
- Карточки с названием, описанием, количеством звёзд и датой обновления
- Фильтрация по организациям через мультиселект (можно выбрать несколько)
- Поиск по названию репозитория
- Сортировка по алфавиту
- Пагинация на десктопе с отражением текущей страницы в URL (`?page=`)
- Бесконечный скролл на мобильных устройствах (≤ 414px) с обновлением URL

### Страница репозитория
- Детальная информация о репозитории
- Список языков программирования
- Список авторов

### Прочее
- Индикатор загрузки (Loader)
- Состояние «Ничего не найдено»
- Адаптивная вёрстка (mobile / tablet / desktop / wide)
