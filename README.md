# GitHub Client

SPA-приложение для просмотра публичных репозиториев GitHub-организаций

## Функциональность

- Список репозиториев организации, загруженных через GitHub API
- Карточки репозиториев: название, описание, количество звёзд, дата обновления
- Клиентская пагинация — каждая страница подгружает свой набор данных через URL-параметр `?page=`
- Адаптивная вёрстка (mobile / tablet / desktop)
- Переход на страницу отдельного репозитория (в разработке)

## Стек
- React
- TypeScript
- Vite
- SCSS

## Статус

Проект в разработке. Реализована страница со списком репозиториев, страница отдельного репозитория в разработке
