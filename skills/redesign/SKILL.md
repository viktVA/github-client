# Скилл: Редизайн через variables.scss

## Что делать

1. Открыть `src/components/variables.scss` — это единственный файл для изменений
2. Если есть компоненты которых нет и они часто используются - добавить компонент в папку `src/components/`
3. Если есть компоненты, которые используются только в пределах одного компонента, то добавлять в папку `Component/components/`, например если используется только на странице RepoListPage то путь `src/App/pages/RepoListPage/components/`
4. Если есть референс в `design-reference/` — разобрать его цвета, шрифты, радиусы, тени, отступы, компоненты
5. Обновить значения в variables.scss под референс
6. Классы компонентов НЕ трогать
7. Props-интерфейсы НЕ трогать
8. Запустить `npm run typecheck`
9. Если меняется только цвет компонента, то менять только значение переменной в файле variables.scss


## Чего не делать
- Не хардкодить значения напрямую в `.module.scss` файлах
- Не добавлять новые классы, если не создаешь новый компонент
- Не менять структуру переменных — только значения
